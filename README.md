# Social-Media-App-Clone
 An Social Media App Clone Inspired by Instagram


## Install Node Modules
In Terminal <br />
cd ./frontend <br />
npm -i <br />
cd .. <br />
cd ./backend <br />
npm -i <br />

Also you need to introduce the database by opening a dotnet folder to the backend.

## Run The App
In Terminal <br />
cd ./backend <br />
npm run dev <br />
Then open a new terminal <br />
cd ./frontend <br />
npm run start <br />
